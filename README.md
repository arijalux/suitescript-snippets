## üöÄ Getting Started

Follow these steps to install and run suitescript-snippets on your computer:

1. **Install Visual Studio Code**: If you do not have Visual Studio Code installed, please download it from the [official website](https://code.visualstudio.com/). Follow the instructions on their site to complete the installation.

2. **Visit the Release Page**: Go to our [Releases page](https://github.com/arijalux/suitescript-snippets/releases) to find the latest version of suitescript-snippets.

3. **Download the Snippets**: Look for the latest release, and select the appropriate file to download. 

4. **Add Snippets to VS Code**:
   - Open Visual Studio Code.
   - Click on the gear icon in the lower-left corner and select "Extensions".
   - Search for "suitescript-snippets" in the extensions marketplace.
   - Click "Install" once you find the extension.

5. **Using the Snippets**:
   - Open a new or existing JavaScript file in VS Code.
   - Start typing the keyword for any script snippet (You can view the list in the extension's documentation).
   - Press `Tab` to insert the snippet into your code.
